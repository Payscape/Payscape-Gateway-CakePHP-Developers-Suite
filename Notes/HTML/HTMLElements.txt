LINK
 HtmlHelper::link(string $title, mixed $url = null, array $options = array(), string $confirmMessage = false)


    Parameters: 

        $title (string) – The text to display as the body of the link.
        $url (mixed) – Either the string location, or a routing array.
        $options (array) – An array of html attributes.


General purpose link 

echo $this->Html->link('Enter', '/pages/home', array('class' => 'button', 'target' => '_blank'));



Use 'full_base'=>true option for absolute URLs:

echo $this->Html->link(
    'Dashboard',
    array('controller' => 'dashboards', 'action' => 'index', 'full_base' => true)
);

BUTTON

echo $this->Html->link('Next', array('controller'=>'uploadimages', 'action'=>'view', $property['id']), array('class' => 'button', 'target' => '_blank', 'id'=>'next'));



IMG

echo $this->Html->image('cake_logo.png', array('alt' => 'CakePHP', 'height' => '300', 'width' => '180'));

to create an image with the full path, for example for emails, 
use the fullBase option

echo $this->Html->image("logo.png", array('fullBase' => true));

IMG LINK

To create an image link specify the link destination using the url option in $htmlAttributes.:

echo $this->Html->image("recipes/6.jpg", array(
    "alt" => "Brownies",
    'url' => array('controller' => 'recipes', 'action' => 'view', 6)
));
